# 🎵 다크 뮤직 — Dark Music Player

**A simple and beautiful music player for Windows.**

## 📥 Download the installer

Download the `DarkMusic_Setup.exe` file.

## 📦 Installation

1. Run `DarkMusic_Setup.exe`.
2. Follow the installer instructions.
3. After installation, launch the player via the desktop shortcut or the Start menu.

## ✨ Features

- Fast music scanning on your computer.
- Play tracks with convenient controls.
- Favorites – save your loved songs.
- Track repeat and seeking.
- Stylish dark interface.

## 🖥️ System requirements

- Windows 7, 8, 10, 11 (32/64 bit)
- No additional libraries required.
